## Como obter os áudios

Você pode baixar as músicas (ou trechos) de qualquer fonte e renomeá-las conforme a tabela acima.

**Formato**: MP3 (compatível com todos os navegadores mobile).

**Dica**: Use trechos de ~2 minutos da parte mais significativa de cada música para manter o site leve.

## Funcionamento

- A música troca automaticamente com **crossfade suave** ao mudar de seção
- O botão 🔊 no canto inferior direito permite mutar/desmutar
- O áudio é inicializado no toque do botão "play" (necessário para dispositivos móveis)
- Se um arquivo estiver faltando, o site continua funcionando normalmente sem áudio naquela seção
